## Component API

### Props
- `projects: Project[]` - List of projects to display in the grid
- `stats: DashboardStats` - Aggregated statistics for the stats grid
- `activities: Activity[]` - Recent activity items for the feed
- `onProjectClick: (id: string) => void` - Navigation handler for project cards
- `onQuickAction: (action: QuickActionType) => void` - Handler for quick action buttons

### State
Dashboard uses server components for data fetching. Client interactivity is isolated to `QuickActions` (command palette trigger) and `ProjectCard` (hover state, keyboard focus).

### Events
- `onProjectClick` - Navigates to project detail page
- `onQuickAction` - Triggers command palette or navigates to create flows
- `onKeyboardShortcut` - Handles ⌘K, N, and arrow key navigation

## Required UI States

### Loading
Skeleton grid matching the StatsGrid (4 shimmer rectangles) + 6 skeleton ProjectCards + 5 skeleton ActivityFeed lines.

### Empty
Illustration with "No projects yet" heading, subtext "Create your first project to get started", and a prominent "New Project" CTA button.

### Error
Inline error banner at top of dashboard: "Failed to load dashboard data. Please try again." with a Retry button. Individual sections degrade independently.

### Disabled
Quick action buttons show disabled state with tooltip "Requires project owner role" when user lacks permissions.

### Validation
Not applicable — dashboard is read-only with navigation actions.
